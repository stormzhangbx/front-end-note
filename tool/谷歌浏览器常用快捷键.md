# 谷歌浏览器常用快捷键

| 快捷键                                              | 含义                                                                   |
| :-------------------------------------------------- | :--------------------------------------------------------------------- |
Ctrl + N | 打开新窗口
Ctrl + T | 打开新标签页
Ctrl + W | 关闭当前标签
Ctrl+Tab | 切换到下一个标签页
Ctrl+Shift+Tab | 切换到上一个标签页
F5 | 刷新页面
Ctrl + F5 | 强制刷新页面
